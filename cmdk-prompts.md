# Cmd+K Menu Prompt Options

## Option 1: Clean Neutral
Design a pure UI command palette triggered by `Cmd+K` for a modern desktop product. You have no context about the product, so do not invent product logic, app-specific workflows, or feature architecture. Focus only on visual design, hierarchy, spacing, states, and motion.

Requirements:
- centered overlay command menu
- search input at top
- grouped command results
- section headers
- icons on the left
- keyboard shortcuts on the right
- clear hover and active states
- recent items section
- empty state
- subtle background blur behind modal
- polished desktop-native feel, especially macOS

Use realistic placeholder commands such as:
- New Document
- Search Files
- Open Settings
- Invite Teammate
- Toggle Theme

Show:
- default state
- typing/filtering state
- keyboard-selected state
- empty results state

Style direction:
- refined typography
- strong spacing rhythm
- subtle depth
- crisp separators
- elegant shadows
- tasteful motion
- production-ready, not generic

Also define:
- spacing system
- color tokens
- typography scale
- border radius
- shadow system
- motion behavior
